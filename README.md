# 팀글벙글(TeamBean)

## 프로젝트 개요

개발 프로젝트 협업 과정에서 다양한 직군의 팀원들과 소통하며 개발하기 위해서는 많은 협업 도구가 필요합니다.
<br >
팀글벙글은 그런 서비스들을 하나의 공간에서 사용해, 협업을 진행하는 과정에 있어 편리함을 가져다 주기 위해 기획되었습니다.
